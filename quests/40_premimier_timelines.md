## 사용한 단축키
1. ctrl + A : 타임라인 전체 선택
2. ctrl + S : 저장
3. ctrl + C : 복사
4. ctrl + V : 붙여넣기
5. ctrl + 휠 : 타임라인 크기 줄이기
6. C : 자르기
7. V : 선택
